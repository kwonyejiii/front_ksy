CSS 애니메이션.

     요소의 현재 스타일을 다른 스타일로 변화
    적용할 스타일은 이전 프레임에 적용하는 것이 아니라 초기 스타일에 적용.(eg. transform 전 값에)

    @keyframes 애니메이션{
        from 혹은 0%
            처음에 적용할 스타일
    }
    정수% (50%, 70%..80%...){
        중간 지점마다 적용할 스타일
    }
    to 혹은 100%{
        마지막에 적용할 스타일
    }
}
